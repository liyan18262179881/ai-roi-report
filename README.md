# AI 赋能商业计划报告生成器

面向中小电商企业的自助式 AI ROI 评估工具。填公司信息、选赋能方向、填几个真实数字,一键生成一份《AI 赋能 XX 品牌商业计划报告》(可导出 PDF),同时算清**收益(降本+增效)**与**投入(自建 vs 外包)**。

## 快速开始

```bash
# 无需安装、无需构建,直接打开即可
open index.html        # macOS
# 或双击 index.html 用浏览器打开
```

如需本地起服务(避免某些浏览器对本地文件的限制):
```bash
python3 -m http.server 8000
# 浏览器访问 http://localhost:8000
```

## 怎么用

1. 左侧填【公司信息】(品牌名/行业/总人数,均可空)并勾选【赋能方向】(视觉、客服、销售、知识库、HR)。
2. 中间为每个方向填**人数、薪资、业务量**。填不出的可空着(会标"待补充",不计入合计)。
3. 点【生成商业计划报告 →】。
4. 在报告页可【导出 PDF】或【返回编辑】。

## 重要原则

- **不编造数字**:用户没填的一律标"待补充",不计入合计。
- 提效系数(产能倍数、AI 处理率等)按 2026 行业基准内置,用户只填人数/薪资/业务量。
- 所有数字均为"基于输入 + 行业基准"的**估算**,不构成投资承诺。
- 不自动拉取、也不伪造任何真实公司的工商/财务数据。

详见 `CLAUDE.md`。
