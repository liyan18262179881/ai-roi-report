# 2026 数据基准与复核清单

> 所有数字为 2026 年 6 月口径的**公开行业基准**,非任何具体品牌真实数据。API 降价快、薪资会变,建议每隔几个月用 web 搜索复核一次,并同步更新 `index.html`(SC.coef / SC.price / compute() 里 selfBuild 薪资)与本文件。

## 一、大模型 / 生成 API 单价(可在工具"我的视角"调)

| 项 | 默认单价 | 说明 |
|----|---------|------|
| 图片生成 | 0.10 元/件 | 通义万相/即梦级 |
| 客服对话 | 0.012 元/次 | 多轮+RAG,按 Qwen-Plus 级估;DeepSeek/豆包更低 |
| 运营内容 | 0.006 元/条 | 文案/详情/帖子 |
| 知识库查询 | 0.01 元/次 | RAG 单次问答 |

要点:**2026 国产大模型 API 价格较 2025 下降约 80%**,这是中小电商规模化用 AI 的成本前提。
